# 通用设计规范

## 色彩系统

### 品牌色
- **主色**：#1890FF（科技蓝）
- **辅助色**：#52C41A（成功绿）、#FAAD14（警告黄）、#F5222D（错误红）
- **中性色**：#000000、#333333、#666666、#999999、#CCCCCC、#F0F0F0、#FFFFFF

### 色彩搭配原则
- **60-30-10 法则**：主色 60%、辅助色 30%、强调色 10%
- **对比度**：文字与背景对比度至少 4.5:1（WCAG AA 标准）
- **色彩层级**：主要信息用高饱和度，次要信息用低饱和度

---

## 字体规范

### 中文字体
- **标题**：思源黑体 / 阿里巴巴普惠体 / PingFang SC
- **正文**：思源黑体 / 苹方 / Microsoft YaHei
- **数字**：DIN / Helvetica Neue

### 英文字体
- **标题**：Helvetica / Roboto / Arial
- **正文**：Helvetica / Roboto / Arial
- **等宽字体**：Courier New / Monaco（代码展示）

### 字号体系
```
H1: 48px  (大标题)
H2: 36px  (副标题)
H3: 28px  (小标题)
H4: 24px  (正文标题)
Body: 16px (正文)
Caption: 14px (辅助文字)
Small: 12px (最小文字)
```

### 行高
- 标题：1.2-1.5 倍字号
- 正文：1.5-2 倍字号

---

## 间距系统

### 基础单位
- **基准**：8px
- **常用间距**：8px, 16px, 24px, 32px, 40px, 48px

### 边距规范
- **页面边距**：24px（移动端）、40px（PC端）
- **卡片内边距**：16px-24px
- **文字行间距**：8px-16px

---

## 图片规范

### 图片比例
- **16:9**：视频、横版大图
- **4:3**：常规图片
- **1:1**：头像、缩略图
- **3:4**：竖版图

### 图片质量
- **清晰度**：避免模糊、像素化
- **文件大小**：
  - 小图（< 100KB）：缩略图、图标
  - 中图（100-500KB）：列表图、卡片图
  - 大图（500KB-2MB）：详情图、海报
- **格式选择**：
  - JPG：照片、复杂图像
  - PNG：logo、透明背景
  - WebP：web 端优先（兼容性注意）
  - SVG：矢量图标

---

## 圆角规范

### 圆角尺寸
- **小圆角**：4px（按钮、标签）
- **中圆角**：8px（卡片、输入框）
- **大圆角**：16px（模块、容器）
- **圆形**：50%（头像、圆形按钮）

---

## 阴影规范

### 阴影层级
```css
/* 轻阴影 - 悬停状态 */
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);

/* 中阴影 - 卡片、弹窗 */
box-shadow: 0 4px 16px rgba(0, 0, 0, 0.12);

/* 重阴影 - 模态框 */
box-shadow: 0 8px 24px rgba(0, 0, 0, 0.16);
```

## 栅格系统

### 移动端（750px）
- **列数**：12 列
- **间距**：16px
- **边距**：24px

### PC 端（1920px）
- **列数**：24 列
- **间距**：24px
- **边距**：40px
- **最大内容宽度**：1200px

## 动效规范

### 过渡时间
- **快速**：0.15s（切换、悬停）
- **标准**：0.3s（展开、收起）
- **缓慢**：0.5s（页面过渡）

### 缓动函数
```css
/* 标准 */
transition: ease 0.3s;

/* 进入 */
transition: cubic-bezier(0.4, 0.0, 0.2, 1) 0.3s;

/* 退出 */
transition: cubic-bezier(0.0, 0.0, 0.2, 1) 0.3s;
```

---

## 无障碍设计

### 可访问性
- **色盲友好**：不仅依靠颜色区分信息
- **键盘导航**：支持 Tab 键切换焦点
- **屏幕阅读器**：图片添加 alt 文本

### 对比度要求
- 正文文字：至少 4.5:1
- 大文字（18px+）：至少 3:1
- 图标、按钮：至少 3:1

---

## 响应式设计

### 断点
```css
/* 移动端 */
@media (max-width: 768px)

/* 平板 */
@media (min-width: 769px) and (max-width: 1024px)

/* PC 端 */
@media (min-width: 1025px)
```

### 适配策略
- 移动端优先（Mobile First）
- 图片响应式缩放
- 文字大小根据屏幕调整
- 布局从单列到多列

---
